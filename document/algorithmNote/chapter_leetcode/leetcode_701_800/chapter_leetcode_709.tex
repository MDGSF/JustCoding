\section{709. 转换成小写字母}
\label{leetcode:709}

\subsection{题目}

实现函数 ToLowerCase()，该函数接收一个字符串参数 str，
并将该字符串中的大写字母转换成小写字母，之后返回新的字符串。

\textbf{示例 1}：

\begin{verbatim}
  输入: "Hello"
  输出: "hello"
\end{verbatim}

\textbf{示例 2}：

\begin{verbatim}
  输入: "here"
  输出: "here"
\end{verbatim}

\textbf{示例 3}：

\begin{verbatim}
  输入: "LOVELY"
  输出: "lovely"
\end{verbatim}

\subsection{参考题解}

\subsubsection{Python}

\begin{verbatim}
class Solution:
  def toLowerCase(self, str: str) -> str:
    return str.lower()
\end{verbatim}
