\section{518. 零钱兑换 II}
\label{leetcode:518}

\subsection{题目}

给定不同面额的硬币和一个总金额。写出函数来计算可以凑成总金额的硬币组合数。假设每一种面额的硬币有无限个。 

\textbf{示例 1}:

\begin{verbatim}
  输入: amount = 5, coins = [1, 2, 5]
  输出: 4
  解释: 有四种方式可以凑成总金额:
  5=5
  5=2+2+1
  5=2+1+1+1
  5=1+1+1+1+1
\end{verbatim}

\textbf{示例 2}:

\begin{verbatim}
  输入: amount = 3, coins = [2]
  输出: 0
  解释: 只用面额2的硬币不能凑成总金额3。
\end{verbatim}

\textbf{示例 3}:

\begin{verbatim}
  输入: amount = 10, coins = [10]
  输出: 1
\end{verbatim}

\textbf{注意}:

你可以假设：

\begin{verbatim}
  0 <= amount (总金额) <= 5000
  1 <= coin (硬币面额) <= 5000
  硬币种类不超过 500 种
  结果符合 32 位符号整数
\end{verbatim}

\subsection{参考题解，动态规划}

\begin{verbatim}
class Solution:
  def change(self, amount: int, coins: List[int]) -> int:
    dp = [1] + [0 for _ in range(amount)]
    for coin in coins:
      for col in range(coin, amount + 1):
        dp[col] += dp[col - coin]
    return dp[-1]
\end{verbatim}
