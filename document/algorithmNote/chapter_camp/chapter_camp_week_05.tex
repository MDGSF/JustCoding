\subsection{学习总结}

这周学习了动态规划，这个还是挺难的。

动态规划关键点：

\begin{itemize}
  \item 最优子结构 opt[n] = best\_of(opt[n-1], opt[n-2], ...)
  \item 储存中间状态:opt[i]
  \item 递推公式(美其名曰:状态转移方程或者 DP 方程)
\end{itemize}
